 # AI 기반 자격증 학습 앱

## 📋 프로젝트 개요

AI 기반 자격증 학습 앱은 인공지능을 활용하여 자격증 취득을 효율적으로 지원하는 스마트 학습 플랫폼입니다. PDF 기출문제를 자동으로 분석하고, 개인 맞춤형 학습 경험을 제공합니다.

## 🎯 주요 기능

### 1단계: 기본 기능 (MVP)
- **PDF 문제 자동 추출**: 기출문제 PDF를 업로드하면 자동으로 문제/선택지/정답 분리
- **스마트 문제 분류**: AI가 문제를 카테고리별, 난이도별로 자동 분류
- **기본 문제 풀이**: 랜덤 출제 및 채점 기능
- **학습 진도 추적**: 맞힌 문제/틀린 문제 기록

### 2단계: AI 강화 기능
- **개인 맞춤 출제**: 취약 영역 분석 후 맞춤형 문제 추천
- **AI 해설 생성**: GPT 기반 상세한 문제 해설 자동 생성
- **학습 패턴 분석**: 학습 시간, 정답률 등을 분석한 대시보드
- **오답 노트**: AI가 틀린 문제의 유사 문제 자동 추천 (에빙 하우스 망각곡선 알고리즘 적용 시도)

### 3단계: 고급 기능
- **음성 문제 읽기**: TTS로 문제 음성 제공
- **실시간 모의고사**: 실제 시험과 유사한 환경 제공
- **학습 커뮤니티**: 사용자 간 질문/답변 공유
- **다중 자격증 지원**: 여러 자격증 동시 준비 가능

## 🛠 기술 스택

### Backend
- **언어**: Python 3.9+
- **프레임워크**: FastAPI
- **데이터베이스**: PostgreSQL (운영), SQLite (개발)
- **AI/ML**: OpenAI GPT-4, Langchain, scikit-learn
- **PDF 처리**: PyPDF2, pdfplumber
- **인증**: JWT

### Frontend
- **웹**: React.js + TypeScript
- **모바일**: React Native (추후)
- **상태관리**: Redux Toolkit
- **UI 라이브러리**: Material-UI / Tailwind CSS

### Infrastructure
- **클라우드**: AWS (EC2, RDS, S3)
- **컨테이너**: Docker, Docker Compose
- **CI/CD**: GitHub Actions
- **모니터링**: CloudWatch, Sentry

